# CodeArena

CodeArena is a coding practice platform where developers and students can improve their programming skills by solving coding challenges.

## Features

* Coding challenges for beginners to advanced
* Daily practice problems
* Leaderboard system
* Programming tutorials

## Website

https://codearena.xo.je

## Technologies Used

* HTML
* CSS
* JavaScript

## Goal

The goal of CodeArena is to help learners practice coding and improve problem-solving skills through interactive challenges.
